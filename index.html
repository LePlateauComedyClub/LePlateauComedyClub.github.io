Lorsque vous rencontrez un problème où un iframe affiche une ancienne version d'une page, cela peut être dû à plusieurs raisons, notamment le cache du navigateur ou le cache du serveur. Voici quelques étapes pour résoudre ce problème et s'assurer que vous affichez la version la plus récente : ### 1. **Forcer le Rafraîchissement du Cache du Navigateur** Votre navigateur peut avoir mis en cache l'ancienne version de la page. Vous pouvez essayer de vider le cache ou de forcer le rafraîchissement pour voir si cela résout le problème. - **Sur Chrome** : - Ouvrez les outils de développement avec `F12` ou `Ctrl+Shift+I` (Windows/Linux) ou `Cmd+Option+I` (Mac). - Cliquez sur l'onglet "Network". - Cochez "Disable cache" (en haut, sous l'onglet Network). - Rafraîchissez la page en appuyant sur `Ctrl+R` (Windows/Linux) ou `Cmd+R` (Mac). - **Sur Firefox** : - Ouvrez les outils de développement avec `F12` ou `Ctrl+Shift+I` (Windows/Linux) ou `Cmd+Option+I` (Mac). - Allez dans l'onglet "Network". - Cochez "Disable Cache" (en haut, sous l'onglet Network). - Rafraîchissez la page avec `Ctrl+R` (Windows/Linux) ou `Cmd+R` (Mac). ### 2. **Ajouter un Paramètre de Cache Busting à l'URL** Pour éviter que les navigateurs utilisent une version mise en cache, vous pouvez ajouter un paramètre de cache busting à l'URL dans l'iframe. Ce paramètre est généralement un timestamp ou un numéro de version. Cela force le navigateur à recharger le contenu. Assurez-vous que le serveur n'a pas de cache persistant qui pourrait empêcher le chargement de la dernière version. ### 4. **Vérifiez l’URL Directement** Assurez-vous que la nouvelle version est bien publiée et accessible directement via le navigateur sans passer par l’iframe : - Ouvrez un nouvel onglet dans votre navigateur et visitez `https://my.weezevent.com/le-plateau-comedy-club-copy-1`. - Vérifiez si la dernière version est affichée. ### Exemple Mis à Jour du Code HTML avec Cache Busting Voici le code HTML mis à jour pour inclure le paramètre de cache busting : ```html Le Plateau Comedy Club
